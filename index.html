Commonly Used CSS Properties

Property Example Description
color color: red; Sets the text color of an element.
background-color background-color: blue; Sets the background color of an element.
margin margin: 10px; Sets the outer spacing of an element.
padding padding: 20px; Sets the inner spacing of an element.
border border: 1px solid black; Sets the border around an element.
width width: 100px; Sets the width of an element.
height height: 150px; Sets the height of an element.
font-size font-size: 16px; Sets the size of the font.
text-align text-align: center; Aligns the text horizontally within an element.
display display: block; Sets an element's display type.
